# Password Insight

Password Insight is a cybersecurity-focused web application designed to help users evaluate the quality of their passwords beyond traditional strength checks. The application analyzes both password strength and predictability, providing a more complete understanding of password security.

In addition to measuring complexity, the application detects common words, names, dates, keyboard patterns, repeated sequences, and other recognizable structures that can make a password easier to guess. Personalized recommendations are then provided to help users create stronger and less predictable passwords.

## Features

- Password strength analysis
- Password predictability analysis
- Real-time password evaluation
- Detection of common words and phrases
- Detection of dates, names, and common patterns
- Detection of keyboard and sequential patterns
- Personalized security recommendations
- Password visibility toggle
- Animated progress indicators
- Responsive design for desktop and mobile devices

## Technologies Used

- HTML
- CSS
- JavaScript

## Development

The initial codebase was generated with the assistance of GitHub Copilot as part of an AI-assisted development workflow.

Further customizations were implemented manually, including scoring refinements, predictability detection improvements, user interface enhancements, testing, bug fixes, and overall application polishing.
